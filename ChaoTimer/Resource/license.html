Scramblers

The most of the scramblers within this app is from DCTimer of MeigenChou. Thanks!

The 3x3 random state scrambler uses Kociemba's Two-Phase algorithm, and the code is an Objective-C port of Shuang Chen's Java min2phase. The Java code is available at https://github.com/ChenShuang/min2phase

The Square one random state scrambler is also an Objective-C port of Shuang Chen's Java sq12phase. The Java code is available at https://github.com/ChenShuang/sq12phase

Thanks!

Custom Picker View

The Custom Scramble Picker View uses Custom iOS7 AlertView, which is available at https://github.com/wimagguc/ios-custom-alertview. It was imported and modified by Jichao Li.

ShareSDK

The social network function in this app uses ShareSDK, which can be downloaded at http://sharesdk.cn.
